# Workshop: interactive mini-menu to choose an action

**Theoretical Recap** An interactive menu in Python is a control structure that repeatedly presents options to the user, reads their input, and dispatches execution to the appropriate function based on that choice. It combines three concepts you have already practiced: loops (while), conditions (if/elif/else), and functions. The classic pattern is a while True loop that displays a numbered list of options, captures input(), evaluates it, calls the matching function, and breaks when the user chooses to quit. A common mental model: think of the menu as a reception desk — it never closes until the visitor explicitly says goodbye. The loop keeps the desk open; the conditions route each visitor to the right office; the functions are the offices doing the real work. Frequent mistakes to avoid: forgetting to cast input() to int (causing a TypeError on comparison), not handling invalid entries (causing a crash), and placing the menu display outside the loop so it only appears once. Always wrap the int() conversion in a try/except block to catch non-numeric input gracefully. Mini example: def say_hello(): print('Hello, user!') while True: print('1. Say hello 2. Quit') choice = input('Choose: ') try: choice = int(choice) except ValueError: print('Invalid input.') continue if choice == 1: say_hello() elif choice == 2: break else: print('Unknown option.')
