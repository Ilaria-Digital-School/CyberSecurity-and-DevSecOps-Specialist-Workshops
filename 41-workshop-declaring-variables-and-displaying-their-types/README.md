# Workshop: declaring variables and displaying their types

**Theoretical Recap** In Python, a variable is a named container that stores a value in memory. Unlike languages such as C or Java, Python is dynamically typed: you do not need to explicitly declare the type of a variable before assigning a value to it. Python infers the type automatically at runtime. The main built-in types you will encounter in security scripting are: int (integers, e.g., port numbers), float (decimal numbers, e.g., ratios or scores), str (strings of characters, e.g., IP addresses, usernames, URLs), bool (boolean True/False, e.g., flags for vulnerability presence), list (ordered collections, e.g., a list of targets), dict (key-value pairs, e.g., HTTP headers), and NoneType (absence of value). To inspect the type of any variable at runtime, Python provides the built-in function type(). For example: ip = '192.168.1.1' followed by print(type(ip)) will output . A common beginner mistake is confusing the integer 80 with the string '80': these are not the same type and cannot be concatenated directly. Another frequent error is forgetting that Python is case-sensitive, so Port and port are two different variables.
