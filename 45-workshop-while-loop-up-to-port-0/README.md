# Workshop: While loop up to port 0

**Theoretical Recap** A `while` loop repeats a block of code as long as a given condition remains True. Unlike a `for` loop — which iterates over a known sequence — a `while` loop is ideal when you do not know in advance how many iterations are needed, or when the stopping condition depends on a value that changes dynamically during execution. The basic mental model: before each iteration, Python evaluates the condition. If it is True, the block executes. If it is False, the loop stops immediately. The critical point is that something inside the loop must eventually make the condition False — otherwise, you get an infinite loop, which is one of the most common beginner mistakes. Example: ```python port = 1024 while port > 0: print(port) port -= 1 ``` Here, `port` decreases by 1 at each iteration. When `port` reaches 0, the condition `port > 0` becomes False and the loop exits. Notice: port 0 itself is never printed — the loop stops before reaching it. Always ask yourself: what is my initial value, what is my condition, and what changes the variable inside the loop?
